#  Library Management System (Python OOP Project)

A mini console-based application built in Python to manage books, users, and borrowing/returning operations.  
It demonstrates **Object-Oriented Programming (OOP)** concepts and **file handling** in Python.

---

##  Features
- Add and view books
- Borrow and return books
- Data persistence using JSON files
- Demonstrates classes, encapsulation, inheritance, and file I/O
